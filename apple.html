Apple

Software Engineering Intern · Summer 2014 · Cupertino, CA

The summer after my sophomore year, I worked at Apple as a software engineering intern in their Wireless Technologies group. I was quickly assigned to my summer-long project—an automated system to test iOS software that would analyze new changes to the codebase, run multiple tests in parallel, and file corresponding bug reports for any regressions.

As the sole engineer for the project with minimal input from my manager, I was left to my own devices to develop the system myself (pun somewhat intended). Most days I spent my time coding away, trying to restore and run tests on iPhones with each new build email that came in. While my internship didn't end up being what I expected, I still learned a great deal: I picked up and became fluent in Python, and I was able to experience the ins and outs of developing a large software project from scratch.
